Lab 9

Inline CSS

Inline CSS is used to alter one single element. In order to use inline CSS you need to add it to the specific element you want to change. The style element can have any CSS property.

Internal CSS

This attribute is used when one page has a unique style that is different from the other pages.It is defined in the style element in the head.

External CSS

External CSS allows you to change the look of an entire website by using one singular file. Note that each HTML page page must contain a reference to the external file inside the link attribute.
